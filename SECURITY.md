# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any ScalarTek repository, please report it responsibly.

**Do not open a public issue.** Instead, email us at **security@scalartek.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide an initial assessment within 5 business days.

## Supported Versions

Security updates are provided for the latest release of each project. We recommend always running the most recent version.
